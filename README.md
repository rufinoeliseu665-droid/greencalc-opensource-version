# Plataforma Terminal GreenCalc

Sistema de Auditoria Ambiental e Tokenização de Ativos Reais (RWA) voltado para o segmento de Green Fintech.

## Funcionalidades
* Encapsulamento de regras financeiras e metas ecológicas.
* Cálculo automatizado de eficiência de captura de CO2.
* Simulação de execução de Contratos Inteligentes.
* Geração de recibo criptográfico estruturado no terminal.

##  Tecnologias Utilizadas
* Python 3
* Programação Orientada a Objetos (POO)
